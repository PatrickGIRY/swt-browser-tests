# SWT Browser Application

Cette application utilise l'Eclipse Standard Widget Toolkit (SWT) pour afficher un document HTML dans une vue de navigateur.

## Prérequis

- Java 8 ou supérieur
- Gradle

## Installation

1. Clonez ce dépôt.
2. Exécutez `./gradlew build` pour compiler l'application.

## Exécution

Exécutez `./gradlew run` pour lancer l'application.

## Configuration

Modifiez le fichier `build.gradle.kts` pour utiliser la bonne version de SWT pour votre système d'exploitation.
